# Fraud Streamlit Dashboard

This project is a multi-page Streamlit dashboard for analysing fraud patterns in banking transaction data.

## Live App
https://fraud-dashboard-adeyemi.streamlit.app/

## Features
- Home dashboard with KPI metrics
- Fraud Insights page
- Risk Factors page
- Data Explorer with filters
- Interactive Streamlit layout

## Dataset
FraudShield_Banking_Data.csv

## Main Technologies
- Python
- Streamlit
- Pandas

## Purpose
This dashboard supports explanatory data analysis by helping users identify fraud patterns, transaction risks, and business-relevant insights from transaction records.
